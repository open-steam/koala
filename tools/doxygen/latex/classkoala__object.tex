Returns a koala\_\-$\ast$ instance matching the given steam\_\-$\ast$ object. You can use this function if you don't know which type a steam object is, e.g. to be able to call specialized functions like \hyperlink{classkoala__object_a3cb2dd24644a08383e0112ca6e5369dc}{get\_\-display\_\-name()}.

Note that this function will at least query the object's type and its OBJ\_\-TYPE attribute. Extensions may also query additional attributes, so this function might be costly regarding server requests. Only use it when you need a matching koala object (e.g. for \hyperlink{classkoala__object_a3cb2dd24644a08383e0112ca6e5369dc}{get\_\-display\_\-name()}) and you cannot determine the matching class yourself.
