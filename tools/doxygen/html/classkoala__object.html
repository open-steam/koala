static koala_object::get_koala_object ( $  steam_object,
$  vars = array() 
) [static, final]

Returns a koala_* instance matching the given steam_* object. You can use this function if you don't know which type a steam object is, e.g. to be able to call specialized functions like get_display_name().

Note that this function will at least query the object's type and its OBJ_TYPE attribute. Extensions may also query additional attributes, so this function might be costly regarding server requests. Only use it when you need a matching koala object (e.g. for get_display_name()) and you cannot determine the matching class yourself.

Parameters:
Object$steam_objectthe steam_* object for which to return a matching koala_* wrapper object
Returns:
Object a koala_* wrapper object that matches the given steam_* object in class
