koala_extension::is_enabled ( $  koala_object = FALSE ) [final]

Checks whether the extension is enabled. If you pass a koala_object as a parameter, then the function will check whether the extension is enabled for that object. Otherwise it will check whether the extension is enabled globally.

You cannot override this function, but you can override is_enabled_for() in your extension to specify whether it is enabled or disabled for a given object.
