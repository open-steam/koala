constructor of steam_connector:

The functions initializes some variables through init() when envoked. Arguments are all optional. If arguments are given, steam_connector tries to connect to the defined sTeam-server. Examples:

  • $steam = new steam_connector( "steam.upb.de", 1900, "aroth", "secret" );
Parameters:
string$pServerIPIP or hostname of a sTeam-server
integer$pServerPortserver's port for COAL-protocol
string$pLoginuser's login
string$pPassworduser's password

method: send_mail($target, $subject, $message, $buffer = 0)

send an email using the smtp module of the server directly. the message send by this command will be delivered to the target address directly and will not be stored in the messaging system of the steam server.

Parameters:
string$targetstring containing the email- address of the recipient
string$subjectsubject of the message
string$fromstring containing the email address of the sender
string$messagemessage body/content in the specified mime type (html by default)
string$mime_typemime type of the message to send e.g. "text/plain". If not specified explicitely the mime type is "text/hmtl" by default
boolean$bufferoptional TRUE if command should be buffered or FALSE to sent immidiately
