function add_annotation:

Adds an annotation to an steam_object

Keep in mind that you can use every subclass of steam_object as an annotation, e.g. steam_document, steam_messageboard or steam_docextern

Examples:

$new_annotation = steam_factory::create_textdoc( $my_object->steam_connector, "My annotation headline", "This is my annotation body. " . "This can be a whole text." ); $my_object->add_annotation( $newAnnotation ); $new_url = steam_factory::create_docextern( $my_object->steam_connector, "Link to the PHPsTeam-Homepage", "http://www.phpsteam.org" ); $my_object->add_annotation( $new_url );

Parameters:
steam_object$pAnnotationObject, which is the annotation
steam_objectobject, which should act as the root (optional)
boolean$pBuffersend command now = 0, send later = 1
Returns:
boolean|integer depends on the buffer argument: boolean if buffer = 0, unique transaction id otherwise

steam_object::get_annotations_filtered ( $  pFilters = array(),
$  pSort = array(),
$  pOffset = 0,
$  pLength = 0,
$  pMaxDepth = 0,
$  pBuffer = FALSE

Returns the annotations of this object, optionally filtered by object class, attribute values or pagination. The description of the filters and sort options can be found in the filter_objects_array() function of the "searching" module of open-sTeam.

Example: Return all the annotations that have been created or last modified by user "root" in the last 24 hours, recursively and sorted by modification date (newest first) and return only the first 10 results: get_inventory_filtered( array( // filters: array( '-', '!class', CLASS_DOCUMENT ), array( '-', 'attribute', 'DOC_LAST_MODIFIED', '<', time()-86400 ), array( '+', 'function', 'get_creator', '==', steam_factory::get_user( $GLOBALS['STEAM'], 'root' ) ), array( '+', 'attribute', 'DOC_USER_MODIFIED', '==', steam_factory::get_user( $GLOBALS['STEAM'], 'root' ) ), ), array( // sort: array( '>', 'attribute', 'DOC_LAST_MODIFIED' ) ), 0, 10 );

Parameters:
pFilters(optional) an array of filters (each an array as described in the "searching" module) that specify which objects to return
pSort(optional) an array of sort entries (each an array as described in the "searching" module) that specify the order of the items
pOffset(optional) only return the objects starting at (and including) this index
pLength(optional) only return a maximum of this many objects
pMaxDepth(optional) max recursion depth (0 = only return annotations of this object)
Returns:
an array of objects that match the specified filters, sort order and pagination.

function get_object_class:

Queries and returns the object's type according to a bitmask of CLASS_* constants. In contrast to get_type(), this can be a mixture of several CLASS_* types, whereas get_type() only returns the major type.

See also:
steam_types.conf.php

Example: $user->get_type() == CLASS_USER $user->get_object_class() == CLASS_USER|CLASS_CONTAINER|CLASS_OBJECT

Returns:
int CLASS_* constant bitmask for this object
