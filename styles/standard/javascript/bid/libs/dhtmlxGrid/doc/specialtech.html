dhtmlXGrid Special Technique

Browser caches xml files

This is native behaviour of IE. IE heavily caches XML files. It never loads it from server again if it has the file with same URL in cache.

Possible solitions:
  • dhtmlx native method (you need to use latest dhtmlxCommon.js and component main js file)
    • mygrid.preventIECashing(true);
  • You can manually add some random element to XML URL
    • mygrid.loadXML("my.xml"); //replace with mygrid.loadXML("my.xml?uid="+(new Date()).valueOf());
