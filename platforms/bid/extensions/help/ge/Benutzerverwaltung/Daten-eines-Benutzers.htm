Daten eines Benutzers

Außerdem können die über den Benutzer gespeicherten Daten geändert werden. Im ersten Feld steht der vollständige Name des Benutzers, in zwei weiteren Feldern der Vor- und der Nachname separat. Diese Felder sind unabhängig voneinander und sollten insbesondere bei Benutzern mit Umlauten oder Leerzeichen im Namen unmittelbar korrigiert werden, nachdem der Benutzer angelegt wurde. Nähere Informationen zur Generierung der einzelnen Namensbestandteile finden Sie im Abschnitt »Benutzer anlegen«. Die Benutzerkennung kann nicht geändert werden; sie wird automatisch generiert.

Ein Schuladministrator kann einen Benutzer zu einer anderen Schule innerhalb des Kreises bzw. innerhalb Bielefelds zuordnen, wenn dieser die Schule wechselt. Wechselt der Benutzer zu einer Schule außerhalb des Kreises, setzen Sie sich bitte mit den eingangs genannten Ansprechpartnern in Verbindung. Diese Funktion ist bislang nicht ausführlich getestet worden. Sollten hier Probleme auftreten, setzen Sie sich bitte ebenfalls mit den Ansprechpartnern in Verbindung.

Für jeden Benutzer kann eine E-Mail-Adresse eingetragen werden, die lediglich zu internen Zwecken der Benutzerverwaltung dient und von den Benutzern selber nicht geändert werden kann. Wenn Sie nach Benutzern suchen, wird diese Adresse angezeigt, so dass einzelnen Benutzern aus der Benutzerverwaltung heraus einfach eine E-Mail zugesandt werden kann. Wir planen, auch den Versand von E-Mails an eine ganze Benutzergruppe zu ermöglichen. Falls Sie hier eine E-Mail-Adresse eintragen, sollten Sie dies dem entsprechenden Benutzer mitteilen, da es sich hier nicht um ein für die Benutzerverwaltung benötigtes Datum handelt und der Benutzer der Speicherung dieser Adresse jederzeit widersprechen kann

Die Rolle eines Benutzers sowie seine Hauptgruppe können nachträglich jederzeit geändert werden. In die Hauptgruppe »Andere Schule« sollten Sie Benutzer eintragen, die Ihre Schule verlassen haben. Über die Funktionen zur Zuordnung von Benutzern können Sie anschließend den Benutzer bei Bedarf aus allen Gruppen an Ihrer Schule entfernen. Nähere Informationen zu den einzelnen Rollen sowie zu Gruppen finden Sie im Abschnitt »Benutzer anlegen«, zur Mitgliedschaft in Gruppen im Abschnitt »Mitglieder zuordnen«.

Beachten Sie auf jeden Fall die Rückmeldung, ob die Änderung des Kennworts bzw. der Benutzerdaten erfolgreich war.
