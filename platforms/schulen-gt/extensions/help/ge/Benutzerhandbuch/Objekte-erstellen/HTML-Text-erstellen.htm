Einen HTML-Text erstellen

Sie können auf dem Server HTML-Texte direkt erstellen, ohne dass Sie diese vorher in einer Datei auf Ihrem Computer speichern müssen. Um einen solchen Text zu erstellen, wählen Sie im Menü »Neu« den Punkt »HTML-Text«. Es öffnet sich das folgende Fenster:

Im Feld »Titel« geben Sie einen Namen für den HTML-Text an, unter dem dieser später im Dokumentenbereich zu sehen ist. Sie können diesen Titel frei wählen.

Sie können im Titel auch einige wenige HTML-Befehle verwenden, um das Aussehen des Titels in den Ordnerübersichten anzupassen. Erlaubt sind die Befehle <b> und <strong> für fette Schrift, <em> und <i> für kursive Schrift, <big> und <small> für kleine bzw. große Schrift sowie <font> für weitere Schriftformatierungen, die Sie per Stilangaben festlegen können. Nähere Informationen zu Stilangaben finden Sie beispielsweise auf den Seiten von SelfHTML. Weitere HTML-Befehle sind aus Sicherheitsgründen nicht zulässig.

Im Feld »Text« können Sie mit Hilfe des in den Server integrierten Editors Ihren Text formatieren. Dazu benötigen Sie keinerlei Kenntnisse über HTML.

Mit einem Klick auf den Button »Text einfügen« wird der Text in den Ordner eingefügt. Beachten Sie, dass der Text erst gespeichert wird, wenn Sie auf diesen Button klicken. Sollte der Browser zwischenzeitlich abstürzen, sind alle bislang vorgenommenen Eingaben in das Textfeld verloren. Sie sollten auf diese Weise daher nur kurze Texte direkt auf dem Server erstellen, um einen möglichen Verlust Ihres Textes in Grenzen zu halten.
