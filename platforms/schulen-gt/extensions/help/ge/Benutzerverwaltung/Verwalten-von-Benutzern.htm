Verwalten von Benutzern

Sie können Daten anderer Benutzer Ihrer Schule einsehen und ggf. modifizieren. Geben Sie in das Formular in dem entsprechenden Bereich einfach einen Teil des gesuchten Namens oder des Benutzernamens ein; es werden dann alle Benutzer angezeigt, deren Name bzw. Benutzername mit der entsprechenden Zeichenfolge beginnt. Wenn die eingegebene Zeichenfolge mit einem Stern (*) beginnt, werden alle Benutzer angezeigt, in deren Namen bzw. Benutzernamen die Zeichenfolge irgendwo enthalten ist. Wenn Sie alle Felder leer lassen, werden alle für Sie sichtbaren Benutzer angezeigt. Es sind jedoch nicht alle Benutzer des Servers für Sie sichtbar, sondern im Regelfall lediglich Benutzer Ihrer eigenen Schule.

Ausgegeben wird eine Liste aller Benutzer, auf die die eingegebenen Suchkriterien zutreffen. Je nach Anfrage und Anzahl der Benutzer an Ihrer Schule kann es einen Moment dauern, bis das Suchergebnis angezeigt wird. Durch einen Klick auf einen der Buchstaben oberhalb des Suchergebnisses können Sie die Anzeige so einschränken, dass nur die Benutzer sichtbar sind, deren Nachname mit dem entsprechenden Buchstaben beginnt.

Angezeigt werden für die einzelnen Benutzer der Name, der Benutzername, die Hauptgruppe, die Rolle sowie ggf. die E-Mail-Adresse. Wenn Sie auf den Namen eines Benutzers klicken, sehen Sie die zu diesem Benutzer gespeicherten Daten. Wenn Sie auf die evtl. vorhandene E-Mail-Adresse klicken, können Sie dem Benutzer eine E-Mail schicken, sofern ein E-Mail-Programm auf Ihrem Computer für Sie nutzbar ist.

Schuladministratoren werden hier zusätzlich zwei Buttons angezeigt. Mit dem einen kann das Kennwort des Benutzers geändert werden, ohne das alte Kennwort zu kennen. Sollte ein Benutzer sein Kennwort vergessen haben, kann ihm so ein neues zugewiesen werden. Außerdem kann dem Benutzer so recht einfach unmittelbar, nachdem er angelegt wurde, ein anderes als das automatisch vergebene Kennwort zugewiesen werden. Nähere Informationen zu dem automatisch vergebenen Kennwort finden Sie im Abschnitt »Benutzer anlegen«. Das Kennwort muss auch hier zwei Mal eingegeben werden und sollte nicht kopiert werden.
