Suchen nach Gruppen

Als Schuladministrator können Sie Daten von allen Gruppen an Ihrer Schule einsehen und bearbeiten. Sie können auch Gruppen löschen. Geben Sie in das Formular in dem entsprechenden Bereich einfach einen Teil der gesuchten Beschreibung in das erste Formularfeld ein; es werden dann alle Gruppen angezeigt, deren Beschreibung mit der entsprechenden Zeichenfolge beginnt. Wenn die eingegebene Zeichenfolge mit einem Stern (*) beginnt, werden alle Gruppen angezeigt, in deren Beschreibung die Zeichenfolge irgendwo enthalten ist.

Alternativ können Sie auch in das zweite Formularfeld eine Zeichenfolge eingeben; es werden dann alle Gruppen gesucht, deren Gruppenname mit der entsprechenden Zeichenfolge beginnt. Wenn Sie in beide Felder etwas eintragen, werden nur die Gruppen angezeigt, die beide Suchkriterien erfüllen. Wenn Sie alle Felder leer lassen, werden alle für Sie sichtbaren Gruppen angezeigt. Es sind jedoch nicht alle Gruppen des Servers für Sie sichtbar, sondern im Regelfall lediglich Gruppen Ihrer eigenen Schule.

Sobald Sie den Button »Gruppe suchen« anklicken, wird das Formular im unteren Bereich erweitert. Dort wird dann eine Liste aller Gruppen angezeigt, auf die die eingegebenen Suchkriterien zutreffen. Je nach Anfrage und Anzahl der Gruppen an Ihrer Schule kann es einen Moment dauern, bis das Suchergebnis angezeigt wird. Für jede Gruppe wird die Beschreibung gefolgt vom Gruppennamen in Klammern angezeigt.

Sie können auf die Beschreibung einer Gruppe klicken. Daraufhin werden unten auf der Seite die Daten der Gruppe angezeigt. Zusätzlich zur Beschreibung und dem Gruppennamen wird auch die interne Gruppennummer angezeigt, die lediglich bei Problemfällen benötigt wird und dann ggf. von den Administratoren unseres Servers von Ihnen erfragt wird. Der Eigentümer der Gruppen wird ebenfalls angezeigt. Dieses Feld ist nur bei automatisch von uns erstellten Gruppen leer. Außerdem wird das Kürzel der Schule angezeigt, der diese Gruppe zugeordnet ist. Im Regelfall wird hier das Kürzel Ihrer Schule stehen.

Außerdem wird eine Liste aller Mitglieder der Gruppe mit deren Namen und Benutzerkennung angezeigt. Wenn Sie auf den Namen eines Benutzers klicken, erhalten Sie die zu dem Benutzer gespeicherten Daten. Mit den beiden am unteren Rand der Seite befindlichen Buttons können Sie zum einen die Gruppendaten bearbeiten, zum anderen eine E-Mail an alle Mitglieder versenden, für die eine gültige E-Mail-Adresse eingetragen ist.

Nach einem Klick auf den Button »Gruppendaten bearbeiten« werden die Daten der Gruppe in einem Formular angezeigt, in dem die einzelnen Daten geändert werden können. Im ersten Feld wird die Beschreibung der Gruppe angezeigt, die bei der Erstellung der Gruppe automatisch erzeugt wurde. Da Lehrer diese nicht bearbeiten können, müssen sie sich an den Schuladministrator wenden, wenn die automatisch vergebene Beschreibung modifiziert werden soll.

Der Eigentümer im zweiten Feld sollte einen sinnvollen Wert erhalten. Wenn ein Lehrer eine eigene Gruppe erzeugt, ist er dort eingetragen. Falls Sie eine Gruppe für einen anderen Benutzer erzeugen, ist es meist sinnvoll, den Benutzernamen dieser Person dort einzutragen. Sie haben hier außerdem die Möglichkeit, die Gruppe an eine andere Schule innerhalb des Kreises bzw. innerhalb Bielefelds zu übertragen. Eine Gruppe kann auch gelöscht werden.
