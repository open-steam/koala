Hilfe zur Benutzerverwaltung

Die Verwaltung von Benutzern dieses Servers können Sie in einem gewissen Rahmen ab sofort für Ihre Schule selbst vornehmen. Falls Sie Schuladministrator sind, müssen Sie die Administratoren des Servers darum bitten, Ihnen eine Benutzerkennung einzutragen und Ihnen die Rolle des Schuladministrators zuzuweisen. Setzen Sie sich dazu bitte mit den Ansprechpartnern in Verbindung. Alle weiteren Benutzer an Ihrer Schule können durch den Schuladministrator eingerichtet werden.

Sie erreichen die Benutzerverwaltung über das Symbol in der Arbeitsleiste des Servers. Dort können Sie sich mit den gleichen Benutzerdaten anmelden, die Sie für den Server verwenden. Welche Funktionen Ihnen zur Verfügung stehen, hängt davon ab, ob Sie als Schüler, Lehrer oder Schuladministrator eingetragen sind. Diese Rolle wurde von Ihrem Schuladministrator festgelegt.

Folgende Funktionen stehen Ihnen zur Verfügung:

Schüler
Eigene Daten einsehen, Kennwort ändern, Benutzer suchen
Lehrer
Eigene Daten einsehen, Kennwort ändern, Benutzer suchen, Gruppe erstellen, Mitglieder zuordnen
Schuladministrator
Eigene Daten einsehen, Kennwort ändern, Benutzer anlegen, Benutzer suchen, Gruppe erstellen, Gruppe suchen, Mitglieder zuordnen

Im Folgenden werden die einzelnen Funktionen der Benutzerverwaltung beschrieben, wie sie derzeit zur Verfügung stehen. In Absprache mit unseren Benutzern wird der Funktionsumfang an Ihre Bedürfnisse angepasst, so dass sich Veränderungen ergeben können, über die wir zu gegebener Zeit informieren werden.
