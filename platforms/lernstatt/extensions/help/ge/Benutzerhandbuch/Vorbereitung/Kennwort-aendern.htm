Kennwort ändern

Sie haben von Ihrem Schuladministrator ein automatisch generiertes Kennwort erhalten. Dieses sollten Sie bei Ihrer ersten Anmeldung aus Sicherheitsgründen sofort ändern.

Klicken Sie dazu auf das Symbol in der Leiste unterhalb des Server-Logos. Sie verbinden sich so mit der Benutzerverwaltung. Dort können Sie sich ebenfalls mit Ihrem Benutzernamen und Kennwort anmelden. Wenn Sie dort »Eigenes Kennwort ändern« anwählen, sehen Sie ein Formular, in das Sie Ihr bisheriges sowie Ihr gewünschtes Kennwort zwei Mal – um die Gefahr eines Tippfehlers zu reduzieren – eingeben müssen. Bestätigen Sie Ihre Eingabe anschließend mit dem Button »Ändern«. Die Änderung des Kennworts wird unmittelbar durchgeführt, so dass Sie sich ab sofort mit dem neuen Kennwort bei allen beteiligten Servern anmelden können.

Bitte beachten Sie die Hinweise oberhalb des Formulars. Der Server für die Benutzerverwaltung verfügt auch über eine eigene Dokumentation sowie eine Online-Hilfe, die weitere Hinweise zur Vergabe von Kennworten enthält. Achten Sie insbesondere auf die exakte Schreibweise; auch Groß- und Kleinschreibung werden unterschieden.
