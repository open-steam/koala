Nutzbare Browser

Um den Server nutzen zu können, benötigen Sie einen aktuellen WWW-Browser; wir raten grundsätzlich dazu, eine möglichst aktuelle Browserversion zu verwenden. Wir testen den Server mit dem Mozilla Firefox (Version 3.0), Opera (Version 9.6) sowie dem Internet Explorer (Version 8.0). Andere Browser sowie ältere Versionen der genannten Browser können zu Problemen führen; bitte aktualisieren Sie daher Ihren Browser bzw. bitten Sie Ihren Systemadministrator dies zu tun. Da sich auf älteren Microsoft-Windows-Systemen keine aktuelle Version des Internet Explorers installieren lässt, sollte dort nach Möglichkeit ein anderer Browser installiert werden.
